## Part B. 발견된 문제점 (취합·우선순위)

심각도는 **[Claude 의견]**: 치명 = 결론의 토대를 흔듦 / 높음 = 결론 해석을 바꿈 / 중간 = 수치 정확도·신뢰도 / 낮음 = 보완.

### B-1. RIM 산식 mis-specification 〔치명〕

**[검증된 사실]** 현재 산식:
```
FV = equity + equity × (adjROE − r) × g / (1 + r − g)
```
분모 `(1+r−g)` ≈ 1.0 (r=8.73%, g clamp [0, 0.0786])이라 보정항 `(adjROE−r)×g/(1+r−g)`가 보통 2% 미만. 따라서 `FV ≈ equity`, **업사이드 ≈ 1/PBR**.

**[검증된 사실]** 수치 예시 (payout=0, r=8.73%):

| 종목 | PBR | adjROE | 현재 업사이드 |
|---|---:|---:|---:|
| A | 0.70 | 10% | 43.0% |
| B | 0.70 | **20%** | 44.1% |
| C | **0.60** | 10% | 66.8% |

→ ROE 2배(10→20%): **+1.1%p** / PBR −0.1(0.7→0.6): **+23.8%p**. **PBR이 ROE보다 약 20배 강하게 작동** = 사실상 저PBR 선별기.

**[Claude 의견]** 원인 추정: Ohlson(1995) 지속성 닫힌형 `B[1+(ROE−r)/(1+r−ω)]`에서 (1) 지속성 ω를 성장률 g로 바꿔치기, (2) 분자에 불필요한 `×g`를 추가로 곱함. 이 `×g`가 값을 짓누름. → 해법은 Part C.

### B-2. 벤치마크 부적합 + 설계서 요구 지표 누락 〔높음〕

**[검증된 사실]** 2025H2~2026H1 KOSPI 급등은 실제이며 극단적 쏠림임:
- 2026-06-18 KOSPI 9,064, 1개월 +24.65%, 전년比 +204.39%.
- 삼성전자·SK하이닉스: 2025-06~2026-05 각각 약 +436% / +1000% 상승.
- 상위 4종목(삼성전자·SK하이닉스·삼성전자우·SK스퀘어) 지수 비중 38.83%(1월)→49.49%(5월).
- 2026-05-06 KOSPI +6.45% 폭등한 날 상승 200종목 / 하락 679종목.

**[Claude 의견]** 동일가중 소형 가치주 20종목 전략을 "사실상 2종목 베팅"이 된 시총가중 지수와 비교해 알파를 매기는 것은 부적합.

## Part C. RIM 산식 재정의 (권고)

### C-1. 메인 권고 — Ohlson(1995) 지속성 단일단계형

```
FV          = equity × [ 1 + (adjROE − r) / (1 + r − ω) ]
FV_per_share = FV / shares
upside       = FV_per_share / price − 1
```
- `ω` = 초과이익 지속성 [0, 1). `equity`(지배주주지분)·`adjROE`(0.5NI+0.5CFO/equity)는 **그대로 유지**.
- **[검증된 사실]** 현재 코드의 최소 수정판 = 분자 `× g` 제거 + `g → ω` 재해석. 분모 구조 유지.
- **[검증된 사실]** 분모 `1+r−ω`는 ω→1에서도 ≈ r(0.087)로 바닥이 막혀 **절대 폭발 안 함** (표준 단일단계 `(r−g)`는 폭발).
- **[검증된 사실]** payout/g 불필요 → KOSDAQ 배당 누락 문제 자동 해소.

**병리 해소 예시 (Claude 계산, ω=0.6):**

| 종목 | PBR | adjROE | 현재 업사이드 | **지속성형 업사이드** |
|---|---:|---:|---:|---:|
| A | 0.70 | 10% | 43.0% | 46.6% |
| B | 0.70 | **20%** | 44.1% | **75.9%** |
| C | **0.60** | 10% | 66.8% | 71.0% |

→ ROE 2배 시 +1.1%p → **+29.3%p**로 회복. 밸류와 퀄리티가 비슷한 크기로 작동.

```python
def rim_fair_value(equity, adj_roe, shares=None,
                   r=0.0873, omega=0.62, vb_cap=5.0):
    """Ohlson(1995) 지속성 단일단계 RIM.
       V = B × [1 + (adjROE - r) / (1 + r - omega)].  omega ∈ [0,1), payout 불필요."""
    if equity is None or equity <= 0:
        return None
    premium = (adj_roe - r) / (1.0 + r - omega)        # V/B - 1
    vb = max(0.0, min(1.0 + premium, vb_cap))           # 세니티 상·하한
    fv_total = equity * vb
    return (fv_total, fv_total / shares) if shares and shares > 0 else fv_total
```

**ω 설정**:
- **[Claude 의견·권고]** 본인 PIT 패널에서 `RI_{t+1} = ω·RI_t + ε` 회귀로 직접 추정(한국·2015~ 기준).
- **[확실하지 않은 사실]** 문헌 prior ≈ 0.6 (예: Dechow-Hutton-Sloan 1999 ω ≈ 0.62) — 미국·1999 데이터이며 정확 수치 검증 권장.
- Phase 2: ω=0.6 고정 → Phase 3 [0.4, 0.9] 베이지안 튜닝.

### C-2. 엄밀화 옵션 — 유한기간 + ROE fade (Phase 3+)

```
FV = B + Σ_{t=1}^{T} (ROE_t − r)·B / (1+r)^t,   ROE_t : adjROE → r 선형 수렴, terminal RI = 0
```
**[Claude 의견]** 가장 방어 가능하나 파라미터(T 등) 증가. Phase 2는 C-1로, Phase 3 멀티모델에서 타입별 비교.

### C-3. 베이스라인 — 무성장 영구형 (세니티용)

```
FV = equity × [ 1 + (adjROE − r) / r ]      # 파라미터 0개
```
**[Claude 의견]** "정교한 모델이 이 0-파라미터 베이스라인을 이기는가" 확인용.

### C-4. 적정 PBR(V/B) 비교 (ROE 15%, r 8.73% 기준)

|---|---:|:---:|
