## 1. 비교 방법론

### 우리 팩터 스크리닝 조건

| 항목 | 조건 |
|------|------|
| 팩터 | GP/A (1/3), 매출YoY (1/6), 영업이익YoY (1/6), 1/PBR (1/3) |
| 점수 | 각 팩터별 퍼센타일 랭크 → 가중합 |
| 필터 제외 | HardFilter · StabilityFilter 미적용 (팩터만 단독 비교) |
| 유니버스 | `universe_gate_pit` PASS 종목 (`stocks.is_excluded=FALSE`에서 SPAC·REIT 제외됨) |
| 데이터 | PIT 기준 — `available_from <= rebalance_date` (룩어헤드 방지) |
| 재무제표 | CFS 우선, OFS fallback / **4월=FY 연간, 8월=H1 반기** |
| 시장 | KOSPI + KOSDAQ |

### DQ Gate REJECT 조건 (자동 제외)
- R02: 자본총계 < 0 (자본잠식)
- R04: 핵심계정(매출액·영업이익·자본총계) 2개 이상 누락
- R09: 자산 ≠ 부채+자본 오차 > 5%

### 팩터 계산에서 자연 탈락
- 금융업: 매출액·영업이익 계정 미수집 → 팩터 null → 퍼센타일 0점
- 우선주 125개 유니버스 포함되지만 점수 낮아 TOP20 미등장
