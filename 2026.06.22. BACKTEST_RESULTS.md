## 2. 전략 파이프라인

```
DQ Gate PASS 유니버스
      ↓
  [Step 1] HardFilter
      ↓
  [Step 2] StabilityFilter (R1~R6)
      ↓
  [Step 3] FactorScreener  ← G/H에서만 사용
      ↓
  [Step 4] MomentumFilter  ← F/G/H에서만 사용
      ↓
  [Step 5] RIMModel (Ohlson 지속성 형태)
      ↓
  Equal-Weight Portfolio (20종목)
```

### Step 2: StabilityFilter

| 코드 | 규칙 | 기준 |
|------|------|------|
| R1 | 부채비율 | 부채/자본 ≤ 200% |
| R2 | 차입금비율 | 차입금/자본 ≤ 150% (3FY 단조 감소 예외) |
| R3 | 매출 역성장 | 3FY 중 YoY < −5% 구간 2회 미만 |
| R4 | 영업CF 연속 음수 | 2년 연속 음수 불가 |
| R5 | 차입 운영 | 영업CF < 0 AND 재무CF > 0 동시 불가 |
| R6 | adjROE | adjROE = (0.5×NI + 0.5×CFO)/equity ≥ 8.73% |

> R6 ablation(D/E/F/G `_no_r6`): R6만 해제, 나머지 R1~R5 유지

### Step 5: RIMModel (Ohlson 1995 지속성 형태)

```
adjROE  = (0.5 × NI + 0.5 × CFO) / equity        [Dechow 1994, λ=0.5]
premium = (adjROE − r) / (1 + r − ω)              [Ohlson 1995, ω=0.62]
V/B     = max(0, min(1 + premium, 5.0))           [세니티 캡]
FV      = equity × V/B
upside  = FV / shares / price − 1  ≥ 5%인 종목만 편입
```

> 이전 버전은 `FV = B + B×(adjROE−r)×g/(1+r−g)` (g=adjROE×payout) 형태였으나,
> adjROE≈r 구간에서 g≈0이 되어 밸류에이션 신호가 소멸하는 결함 수정.

### 6-3. FactorScreener 역효과 ❌

### 6-5. R6의 역할

이상치 구간(2025-08~2026-06)을 제외한 2016-04~2025-08 기준:
