## 2. 전략 파이프라인 (Phase 2 기본 구성 — G_full)

```
DQ Gate PASS 유니버스
      ↓
  [Step 1] HardFilter
      ↓
  [Step 2] StabilityFilter
      ↓
  [Step 3] FactorScreener
      ↓
  [Step 4] MomentumFilter
      ↓
  [Step 5] RIMModel (적정가 계산 + 업사이드 필터)
      ↓
  Equal-Weight Portfolio (20종목)
```

### Step 1: HardFilter

| 규칙 | 기준 |
|------|------|
| 거래유동성 | 최근 20 영업일 일평균 거래대금 ≥ 1억원 |
| 상장기간 | listed_date 기준 6개월 이상 |
| PIT 데이터 | FY PIT 시리즈 존재 여부 확인 |
| 상장폐지 | `stock_listing_events` 기준 당일 상폐 여부 확인 |
| 미구현 | 감사의견 비적정(R06), 관리종목(R08) — Phase 3 이후 추가 예정 |

### Step 2: StabilityFilter (재무안정성 필터)

**하드 룰 6개 — 하나라도 위반 시 탈락 (Bayesian 튜닝 대상 외)**

| 코드 | 규칙 | 기준 |
|------|------|------|
| R1 | 부채비율 | 부채/자본 ≤ 200% |
| R2 | 차입금비율 | 차입금/자본 ≤ 150% (예외: 최근 3FY 단조 감소 + 누적 10%p 이상 개선 시 통과) |
| R3 | 매출 역성장 | 최근 3FY 중 YoY < -5% 구간 2회 미만 |
| R4 | 영업CF 연속 음수 | 영업CF가 2년 연속 음수가 아닐 것 |
| R5 | 차입 운영 | 영업CF < 0 AND 재무CF > 0 동시 불가 |
| R6 | adjROE | adjROE = (0.5×NI + 0.5×CFO)/equity ≥ r(8.73%) |

**참고 플래그 2개 — 탈락 아닌 기록용**

| 코드 | 내용 |
|------|------|
| F1 | 재고자산 회전율 전년 대비 30% 이상 하락 |
| F2 | 매출채권 회전율 전년 대비 30% 이상 하락 |

> R6에서 사용하는 `equity`: 지배기업소유주지분 우선 → 없으면 자본총계 fallback (Dechow 1994 Method C)

### Step 3: FactorScreener

4개 팩터 퍼센타일 랭크 가중합 → 상위 **20%** 선별

| 팩터 | 계산식 | 가중치 |
|------|--------|--------|
| rev_yoy | 매출액 YoY 증가율 | 1/6 (≈16.7%) |
| op_yoy | 영업이익 YoY 증가율 | 1/6 (≈16.7%) |
| gpa | 매출총이익 / 자산총계 (Novy-Marx 2013) | 1/3 (≈33.3%) |
| inv_pbr | 1 / PBR | 1/3 (≈33.3%) |

- 동점은 평균 순위 처리
- None 값은 최하위(0) 처리
- top_pct = 20% (Phase 3 Bayesian 튜닝 범위: 10~40%)

### Step 4: MomentumFilter

MA20/MA60 이중 조건을 **동시** 충족 시에만 제외 (하락 추세 종목 제거)

| 조건 | 내용 |
|------|------|
| 조건 1 | MA20 < MA60이 5 영업일 연속 (confirm_days=5) |
| 조건 2 | MA60(현재) < MA60(20일 전) — 장기 MA 우하향 (slope_lookback=20) |
| 데이터 부족 | lookback 데이터 부족 시 통과 (보수적 포함) |

Phase 2에서 파라미터 4개 (ma_short=20, ma_long=60, confirm_days=5, slope_lookback=20) 모두 고정.

### Step 5: RIMModel (잔여이익모델)

```
adjROE = (0.5 × NI + 0.5 × CFO) / equity        [Dechow 1994, λ=0.5]
g      = adjROE × (1 - payout), clamp [0, r×0.9]
FV     = equity + equity × (adjROE - r) × g / (1 + r - g)
FV_per_share = FV / shares
upside_pct = FV_per_share / current_price - 1
```

| 항목 | 처리 |
|------|------|
| equity 우선순위 | 지배기업소유주지분 > 지배기업소유주지분_1 > 자본총계 |
| payout 누락 | payout=0 처리 (KOSDAQ 소형주 배당 데이터 누락 다수 → 낙관적 편향 허용) |
| β | 1.0 고정 (Phase 3 이후 rolling β 도입 검토) |
| rim_threshold | 업사이드 5% 이상인 종목만 편입 (Phase 2 기본값) |

### 4-3. 판정 결과

| 판정 기준 | 결과 | 수치 비교 |
|---------|:----:|---------|
| C > B (재무안정성 기여) | ✅ 통과 | 5.87% > 3.50% (중앙값) |
| D > C (RIM 유효성) | ✅ 통과 | 11.09% > 5.87% (중앙값) |
| E > D (팩터 스크리닝 기여) | ❌ 미통과 | 3.53% < 11.09% |
| F > D (모멘텀 기여) | ❌ 미통과 | 10.40% < 11.09% |
| G > D (전체 필터 기여) | ❌ 미통과 | 6.89% < 11.09% |

**핵심 해석:**
- RIM은 유효하다 (D가 C 대비 +5.2%p CAGR 향상)
- FactorScreener가 현재 파라미터에서 **역효과** — D 대비 -7.6%p CAGR 손실
- MomentumFilter 단독 추가는 D 대비 소폭 저하 (-0.7%p)
- 전체 필터(G)는 D보다 오히려 나쁨 → FactorScreener 문제가 주원인으로 추정
- StabilityFilter 제거(H)는 MDD 크게 악화 (-44.48% vs D -31.66%)

## 8. 핵심 발견 및 해석

### 8-1. RIM 유효성 (D > C) ✅

HardFilter + StabilityFilter 후 무작위 선택(C: 중앙값 5.87%)보다 RIM으로 선택(D: 11.09%)이 +5.2%p CAGR 우월. **RIM은 유효한 선별 신호를 제공한다.**

### 8-2. FactorScreener 역효과 (E < D) ❌

FactorScreener 추가 시 CAGR 11.09% → 3.53%로 급락. 원인 후보:
- StabilityFilter 통과 후 고품질 종목 중에서 팩터 기준으로 재필터링 시 **RIM이 주목하는 저PBR + 높은 FCF 종목이 탈락**
- gpa(1/3), inv_pbr(1/3) 가중치가 과도하게 높아 이미 StabilityFilter가 걸러낸 속성과 중복 or 충돌
- 가중치 파라미터 튜닝 필요 (Phase 3 Bayesian 최적화 대상)

### 8-3. MomentumFilter 소폭 저하 (F < D) ❌

-0.7%p CAGR 차이. 하지만 MDD -31.66% → -29.65%로 **리스크는 소폭 개선**. Momentum 기여도는 CAGR 기준 미미하나 MDD 관리에서 약간의 효과.

### 8-4. StabilityFilter 기여 (D vs H) 명확

StabilityFilter 없애면 (H: 8.09%, MDD -44.48%) vs 있으면 (D: 11.09%, MDD -31.66%):
- CAGR -3%p, MDD +12.8%p 악화
- 특히 #10 구간(코로나 직전, 2019-08-20): H -29.16% vs D -23.89% 차이 확인
- **StabilityFilter는 Value Trap 방어에 효과적**

### 8-5. 최근 2개 구간의 구조적 문제

#22 (2025-08-20~2026-04-03)과 #23 (2026-04-03~2026-06-20)에서 KOSPI가 각각 +71.8%, +68.4% 급등하는 동안 전략 전체가 최대 +25%대에 머묾. 이 두 구간이 최종 Alpha를 전부 음수로 견인하는 주원인. 

원인 추정:
- 해당 기간 KOSPI 급등이 대형주/성장주 중심으로 집중
- 본 전략은 소형 가치주(저PBR, 고FCF) 중심으로 이 랠리에서 소외
- 벤치마크로 KOSPI가 아닌 KOSPI 소형주 지수 비교 검토 필요 (Phase 2 이후 과제)

### 8-6. 최적 파이프라인 현재 결론

현재 파라미터 기준: **D_rim_only (HardFilter + StabilityFilter + RIM)** 가 가장 높은 CAGR(11.09%)과 Sharpe(0.411)를 기록. 단순한 파이프라인이 더 좋은 성과를 보이는 상황.

Phase 3 방향:
1. FactorScreener 가중치 Bayesian 튜닝 (gpa/inv_pbr 비율 조정)
